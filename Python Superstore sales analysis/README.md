# Python Superstore Sales Data Analysis

## Project Overview

This project analyzes the Superstore Sales dataset using Python to uncover business insights related to sales performance, profitability, customer behavior, and regional trends. The analysis includes data cleaning, exploratory data analysis (EDA), feature engineering, and data visualization.

## Objective

The objective of this project is to transform raw sales data into meaningful insights that help understand business performance and support data-driven decision-making.

- Jupyter Notebook
