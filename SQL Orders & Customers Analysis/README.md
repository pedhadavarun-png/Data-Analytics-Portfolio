# SQL Orders and Customers Analysis

## Project Overview
This project analyzes orders and customer data using SQL to generate business insights. The analysis focuses on sales performance, customer behavior, and order trends.

## Tools Used
- MySQL

## Key SQL Concepts
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS (INNER,LEFT,RIGHT,UNION)
- Aggregate Functions (SUM, COUNT, AVG)
- Subqueries
- Window functions
- CTE's

## Analysis Performed
- Total number of orders
- Total sales amount
- Top customers by sales
- Sales by category
- Monthly sales trends
- Customer order frequency
