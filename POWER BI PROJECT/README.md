# Power BI HR Dashboard

## Project Overview

This project presents an interactive HR Analytics Dashboard developed using Microsoft Power BI. The dashboard provides insights into employee demographics, attrition trends, job satisfaction, salary distribution, and workforce performance to support data-driven HR decisions.

## Objective

The objective of this project is to analyze employee data and identify key factors affecting workforce performance and employee attrition through interactive visualizations.

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel (Dataset)

## Dashboard Features

- Employee Count Overview
- Attrition Analysis
- Attrition Rate
- Department-wise Employee Distribution
- Gender Distribution
- Age Group Analysis
- Education Field Analysis
- Job Role Analysis
- Salary Distribution
- Job Satisfaction Analysis
- Interactive Filters and Slicers

## Key Insights

- Identified departments with the highest employee attrition.
- Compared attrition rates across different age groups and genders.
- Analyzed salary distribution among employees.
- Evaluated employee distribution by department and job role.
- Examined job satisfaction trends to understand workforce engagement.
